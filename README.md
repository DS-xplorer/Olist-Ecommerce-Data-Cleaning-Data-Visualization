# 🛒 Olist E-Commerce Data Cleaning & Data Visualization

## 📖 Project Overview

This project showcases an end-to-end data cleaning and visualization workflow using the Olist Brazilian E-Commerce dataset. The primary objective was to clean raw data, improve data quality, and build an interactive Tableau dashboard to uncover meaningful business insights.

The dataset was processed using Power Query to handle missing values, duplicates, and inconsistent data before creating an interactive dashboard in Tableau. The final dashboard was also exported as an HTML file for easy sharing and presentation.

---

## 🎯 Objectives

- Clean and prepare raw e-commerce data
- Improve overall data quality
- Build an interactive Tableau dashboard
- Analyze sales performance and customer behavior
- Generate actionable business insights through visualization

## 🧹 Data Cleaning Process

The dataset was cleaned using Power Query by performing the following steps:

- Removed duplicate records
- Handled missing values
- Corrected data types
- Renamed columns
- Removed unnecessary columns
- Standardized text formatting
- Prepared data for visualization
